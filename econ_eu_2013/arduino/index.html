Mélanie Bats / Obeo

Hi, I'm Melanie Bats and I work as software developer at Obeo. I'm here to present you the developments I did to turn the Eclipse IDE into an Arduino programming platform for kids.

Curiosity

Explore, question and learn

Children are such curious creatures. They explore, question, and wonder, and by doing so, learn. From the moment of birth, likely even before, humans are meant to learn new things. When we are curious about something new, we want to explore it.

Discover by yourself

Explore to understand cause and effects.

A good way to discover is to explore by yourself, trying, encountering success and failures. This way, kids learn causes and effects.

Understand programming

Many open source software are dedicated to learn programming to kids : Scratsh, TurtleLogo, Ruby4Kids, Python4Kids... See : Programatoo, Devoxx4Kids...

There are already many existing open source software for kids to learn programming. The most famous, TurtleLogo is for the eldest ones. More recently, Scratch has become more popular. These tools have different approaches.

Text

In TurtleLogo you write text to move a small turtle in order to draw something. No graphical UI, very simple language.

Graphical

In Scratch you drag and drop elements from a palette to animate a character. This tool is a complete IDE which allows kids to create cartoons, video games, movies...

Eclipse IDE for developpers

Complex UI, concepts (perpsectives, project lifecycle...), languages (Java, C)

Eclipse is one of the most famous IDE. It is used by more than xxx persons everyday. So why kids do not use it also to learn electronics and programming? The Eclipse UI and default available languages as Java or C are to complex to discover programation.

Turning Eclipse an IDE for kids

Graphical programming & light UI

The idea was to simplify the UI, the project lifecycle and to provide a graphical block language to be able to develop simple programs and run them on electronic modules.The purpose is not to provide a complete IDE but a tool giving the possibility to discover the effects a software program could have on electronic modules.

A simplified UI

No menu, only a toolbar with 4 buttons activated according to what is possible Dashboard

The UI defines no menu just one toolbar with 4 different buttons to : manage project lifecycle, define the hardware modules used and to define the sketch that would be uploaded on the hardware platform.

Arduino designer : Dashboard

Show the process

      Define connection between hardware platform and modules
      Describe the sketch
      Upload on the target

A dashboard helps the user by showing him what is the next step in the process development. The process is really simple in fact.

Arduino designer : Hardware

Platform, Modules, Wire

First step, the kid defines the hardware platform he is using and the different sensors and actuators he connects to the Arduino.

Sketch

Graphical block-based language

Then he describes the sketch which represents the software that would run on the hardware platform. And finally, he uploads the sketch to the target.

Sirius

A tool to quickly define DSL based custom multi-view workbenches with dedicated representations

To develop the graphical designers I used Sirius. Sirius is a new project in Eclipse to quickly define multi view workbenches based on DSL with dedicated representations.

Why using Sirius ?

  • For the tool development phase : Dynamic & iterative (live) development
  • For kids :
    • Tooling adapted to the development of arduino small projects
    So why am I using Sirius? During the prototype development phase, it allows me to develop really quickly an arduino specific modeler. The development is easy, dynamic and iterative thanks to a live preview. I choose it also because I am used to work with Sirius in my daily work as I use it also to develop another software UML Designer. For the kids, they get a tooling completely adapted to the development of small arduino projects and really simple to use.

    First step in the development of the arduino designers : I defined a new DSL specific to Arduino which allows me to describe Arduino hardware and software sketches.
